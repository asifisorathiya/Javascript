What are Operators?

Operators are used to perform certain operation on operands and values for achieving final results, ex: 2+3, 2 and 3 are operands and + is an operator
There are following types of Operators as follows:
  • Arithmetic: +,-,*,/,%,++,--
  • Comparison: ==, !=,===,!==,<,> <=,>=
  • Logical:&&,||, !
  • bitiwse:&,|,~,^
  • Assignment:=
  • Miscellanous Operator:Conditional,??,spread, yeild

Grouping Operator

Grouping operator is used to group certain operation. ex: 5+6*10 we can group this as (5+6)*10

Yield Operator

Yeild operator is used to pause and resume the generator function. in simple words use next() method to run the function by resume it. ex: function* test() { yield 10; yield 30; } const t=test(); t.next();
