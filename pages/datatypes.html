There are 2 types of Data types 1.) Primitive 2.) Object

Primitive Data Types

  • There are 7 types of Primitive data types
  • Number: Number is stored
  • String: value stored in qoutes
  • Boolean: Value stored in true or false
  • Symbol: value is unique and cannot be changed
  • Undefined: value is empty declared not used
  • null: value is null
  • bigint: value stored in large Number introduced in es2020
