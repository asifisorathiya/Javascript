What is javascript?


  • JavaScript is a lightweight open source and cross platform programming language which is used to create dynamic and interative elements with web applications
  • The advantages of javascript are Client Side validation, manipulationg html pages, data rendering, data visualization.

JavaScript has two types of values 1.) Fixed Values (Literals) 2.) Dynamic values (var)

Data Types

JavaScript support primitive and non primitive data types
Data types are the base of programming languages since it stores the values into variables.
Primitve data types are string,number,boolean,null,undefined in ES5,Symbol and Bigint introduced in ES6.
Composite Data types are Object, Array & Date
Object is in the json format which stores key value pairs.
Array stored the information in the list format
Date is a simple date.
typeOf is used to check the data type of the variable.

strict Mode

'use strict' means variable declaration is must and readable property cannot be changed.
Introduced in ES5 2009 to error prevention and security reasons.
